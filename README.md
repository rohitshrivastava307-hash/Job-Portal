# Job Portal

A full-stack Job Portal Application built using Angular, Node.js, Express.js, and MongoDB.

## Features

### Authentication

* User Registration
* User Login
* JWT Authentication
* Role-Based Authorization

### Recruiter Features

* Create Jobs
* Edit Jobs
* Delete Jobs
* View Applicants
* Accept / Reject Applicants

### Job Seeker Features

* Browse Jobs
* Apply for Jobs
* Track Application Status

### UI Features

* Responsive Design
* Toast Notifications
* Loading States
* Error Handling
* Empty States

## Tech Stack

### Frontend

* Angular 16
* Tailwind CSS
* TypeScript

### Backend

* Node.js
* Express.js
* MongoDB
* JWT

## Installation

### Frontend

```bash
cd frontend
npm install
ng serve
```

### Backend

```bash
cd Backend
npm install
npm start
```
